# OKR Tracker for Jira Legal Docs

Public legal and Marketplace review documents for OKR Tracker for Jira.

Recommended GitHub Pages setup:

1. Repository Settings -> Pages
2. Source: Deploy from a branch
3. Branch: `main`
4. Folder: `/docs`

Once Pages is enabled, publish these URLs in Atlassian Marketplace:

- Privacy Policy: `/privacy-policy.html` or `/privacy-policy`
- End User Terms: `/terms-of-use.html` or `/terms-of-use`
- Support: `/support.html` or `/support`

This repository intentionally contains only legal/support/public-review content and no application source code.
